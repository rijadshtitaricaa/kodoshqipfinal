Mësimi 1: Hyrje dhe Komanda `main()`

Java është një gjuhë programimi me qëllim të përgjithshëm, e orientuar drejt objekteve dhe e bazuar në klasa. Ajo është projektuar të jetë e varur sa më pak nga implementimi, duke ndjekur parimin "Shkruaje një herë, ekzekutoje kudo" (WORA). Komanda kryesore e ekzekutimit është `main`.

Kodi Juaj i Parë në Java: `main()`

Funksioni **`System.out.println()`** shfaq daljen në konsolë. Zakonisht është funksioni i parë që mëson çdo kush. 
public class Hyrje {
    // Pika e hyrjes së programit
    public static void main(String[] args) {
        // Funksioni për shfaqjen e tekstit në konsolë
        System.out.println("Përshëndetje, Bota!");
    }
}
